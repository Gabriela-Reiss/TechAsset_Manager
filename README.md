## TechAsset Manager

O TechAsset Manager é um sistema web desenvolvimento para gerenciar ativos de tecnologia
de uma empresa, substituindo controles manuais.

Por meio desse sistema é possível organizar por categorias e manter registro de informações 
relevantes sobre os ativos.

### Tecnologias
- ASP.NET CORE MVC
- Entity Framework Core
- Bootstrap
- SQL Server
